a) var z; alert(typeof z);

Result: It pops up and says "undefined".
Why: Because there is no value in z.

b)alert(!!false);

Result: It pops up and says "false".
Why: Because "!" means "not". "!false" = not false = true, "!!false" = not not false = false.

c) alert(7%2);

Result: it pops up and says "1"
Why: the "%" will show the remainder.

d) var name="Jim"; alert(name+" Smith");

Result:Pops up and says "Jim Smith" Why: because the variable name was assigned the string value of Jim.

e) var myNum="5"; alert(myNum+2);

Result: 52 Why: Because the variable myNum was assigned the "string" value "5", which was treated as a string value instead an integer. Therefore, 5+2 will not be calculate as 7 and will be presented as 52.
