a) var z; alert(typeof z);

Result: It pops up and says "undefined".
Why: Because there is no value in z.

b)alert(!!false);

Result: It pops up and says "false".
Why: Because "!" means "not". "!false" = not false = true, "!!false" = not not false = false.

c) alert(7%2);

Result: it pops up and says "1"
Why: the "%" will show the remainder.

d) var name="Jim"; alert(name+" Smith");

Result:Pops up and says "Jim Smith" Why: because the variable name was assigned the string value of Jim.

e) var myNum="5"; alert(myNum+2);

Result: 52 Why: Because the variable myNum was assigned the "string" value "5", which was treated as a string value instead an integer. Therefore, 5+2 will not be calculate as 7 and will be presented as 52.

4.Explain the differences between =, ==, and the === operators.

a. The operator "=" means "assign". The variable on the left has a copy of the variable on the right.
b. The operator "==" means "equal to". Different variable types can be accepted as long as the value is the same. E.g. "5" == 5.
c. The operator "===" means "Equal value and type". The left variable‘s value and type has to be totally the same as the right variable. E.g. "5" is not === 5.

5. Describe the differences between two types of loops: while and do while

a. While loops will firstly check the condition, if true, then do the following action. If not, then skip the action and jump to the next code.
b. Do-while loop will do the action first no matter the condition is true or not. And then check the condition, if true, do it again; if false, then jump to the next code.
